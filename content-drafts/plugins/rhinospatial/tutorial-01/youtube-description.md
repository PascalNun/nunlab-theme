## Description Draft

This tutorial walks through the first complete RhinoSpatial workflow inside Rhino and Grasshopper.

RhinoSpatial is a Grasshopper plugin for bringing site context into Rhino: one shared Spatial Context, then aligned WFS vector data, WMS imagery, GeoTIFF files, terrain, LoD2 buildings, OpenStreetMap context, and Google 3D Tiles reference previews.

The plugin is currently in alpha, so some behavior may still change. The Google 3D Tiles viewer is included as a visual reference viewer only and requires your own Google Maps Platform / Map Tiles API key.
