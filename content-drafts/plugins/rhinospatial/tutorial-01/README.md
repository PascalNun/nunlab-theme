# RhinoSpatial Tutorial 01 Drafts

Working folder for the first RhinoSpatial walkthrough transcript, YouTube copy, and website chapter text.

Suggested files:

- `raw-whisper.vtt` or `raw-whisper.srt` for timestamped transcript output.
- `raw-whisper.txt` for plain transcript output.
- `clean-transcript.md` for the corrected transcript.
- `youtube-description.md` for title, description, chapters, and pinned-comment draft.
- `website-chapters.md` for the RhinoSpatial plugin page copy.
- `website-chapters.wp-blocks.html` for the local WordPress block content used by the sandbox page.
- `terminology-notes.md` for repeated wording and technical-name checks.

Use `screenshots/` for source screenshots that should become website chapter images.
